# Proyecto-OAW

### Project Description

Project developed for the course **Web Application Optimization**. A search engine was implemented with RSS feed integration.

- Search engine with RSS feed integration, including snippet generation.
- The site use techniques such as web crawling, text processing, Boolean queries, relevance weighting, and semantic expansion to improve search accuracy and relevance of results.
- Optimized system performance and user experience through the integration of advanced search methods and automated data retrieval processes.
